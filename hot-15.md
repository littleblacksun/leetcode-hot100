## 解法一

**步骤一：矩阵转置**

· 定义：沿 **主对角线（左上→右下** 翻转，行和列互换

· 规则：matrix[i][j] ↔ matrix[j][i]

· 循环技巧：j < i 只遍历对角线左下侧，避免重复交换

**步骤 2：每行水平翻转**

· 定义：沿行**中线左右**翻转

· 规则：row[j] ↔ row[n-1-j]

· 循环技巧：j < n/2 只遍历前半列，后半列自动完成翻转

>原始矩阵
>
>1 2 3
>
>4 5 6
>
>7 8 9
>
>第一步转置后
>
>1 4 7
>
>2 5 8
>
>3 6 9
>
>第二步：水平翻转后
>
>7 4 1
>
>8 5 2
>
>9 6 3

```java
class Solution {
    // 主方法：原地旋转矩阵90度
    public void rotate(int[][] matrix) {
        // 获取矩阵的边长n（因为是n*n方阵，行和列长度相等）
        int n = matrix.length;

        // ==================== 第一步：矩阵转置 ====================
        // 遍历每一行 i
        for (int i = 0; i < n; i++) {
            // 遍历每一列 j，只遍历 j < i 的区域（避免重复交换）
            for (int j = 0; j < i; j++) {
                // 临时变量存储 matrix[i][j] 的值
                int temp = matrix[i][j];
                // 交换 matrix[i][j] 和 matrix[j][i]
                matrix[i][j] = matrix[j][i];
                // 完成转置交换
                matrix[j][i] = temp;
            }
        }

        // ==================== 第二步：每行水平翻转 ====================
        // 增强for循环：遍历矩阵中的每一行
        for (int[] row : matrix) {
            // 遍历每行的前半部分元素（j < n/2 避免重复翻转）
            for (int j = 0; j < n / 2; j++) {
                // 临时变量存储当前位置元素
                int tmp = row[j];
                // 将当前位置元素 和 该行对称位置元素交换
                row[j] = row[n - 1 - j];
                // 完成水平翻转
                row[n - 1 - j] = tmp;
            }
        }
    }
}
```



## 解法二：一次主体循环  逐行处理 → 转置当前行 + 翻转当前行 一起做完，再下一行

>原始矩阵
>
>1 2 3
>
>4 5 6
>
>7 8 9
>
>逐行i=0
>
>转置：交换(0,1)-(1,0),(0,2)-(2,0)
>
>翻转行0：1 4 7 -> 7 4 1
>
>逐行i=1
>
>转置：交换(1,2)-(2,1)
>
>翻转行1：2 5 8 -> 8 5 2
>
>逐行i=2
>
>转置：无元素可以交换
>
>翻转行2：3 6 9 -> 9 6 3
>
>最终效果
>
>7 4 1
>
>8 5 2
>
>9 6 3

```java
class Solution {
    public void rotate(int[][] matrix) {
        // 获取 n×n 方阵的边长
        int n = matrix.length;

        // 遍历矩阵的每一行 i
        for (int i = 0; i < n; i++) {
            // 把当前行取出来，简化后面代码（等价于 matrix[i]）
            int[] row = matrix[i];

            // ==================== 核心1：矩阵转置 ====================
            // j 从 i+1 开始 → 只遍历【主对角线右上方】的元素
            for (int j = i + 1; j < n; j++) {
                int tmp = row[j];         // 暂存当前位置值
                row[j] = matrix[j][i];    // 把 (j,i) 换到 (i,j)
                matrix[j][i] = tmp;       // 完成交换实现转置
            }

            // ==================== 核心2：当前行水平翻转 ====================
            // 只遍历前半段 j < n/2，避免重复交换
            for (int j = 0; j < n / 2; j++) {
                int tmp = row[j];                     // 暂存左边元素
                row[j] = row[n - 1 - j];              // 右边换到左边
                row[n - 1 - j] = tmp;                // 完成行翻转
            }
        }
    }
}
```
