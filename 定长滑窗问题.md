## 我的思路

移动右指针，并且保持左指针与右指针长度为k，设置一个变量res=0

右指针每一次压入字符，判断是不是元音字母，是的话，res+1，否则不变，左指针每一次弹出字母，判断是不是元音字母，是的话res-1，否则不变，ans记录res的最大

```java
class Solution {
    public int maxVowels(String s, int k) {
        int n = s.length();
        int current = 0;
        int max = 0;

        for(int i = 0;i<k;i++){
            if(isVowel(s.charAt(i))){
                current++;
            }
        }

        max = current;

        for (int right = k; right<n;right++){
            if(isVowel(s.charAt(right))){
                current++;
            }
            int left = right - k;
            if(isVowel(s.charAt(left)))
            {
                current--;
            }
            if (current > max)
            {max = current;}
        }
        return max;
        
    }
    private boolean isVowel(char c){
            return c=='a'||c=='e'||c=='i'||c=='o'||c=='u';
        }
}
```
