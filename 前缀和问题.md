针对于数组不可变的问题，求数组某一个区域范围[left，right]中间的元素值之和

那么可以利用数列和的思想，用一个数组s[i+1](因为初始化s[0]=0)，来记录nums中[0-i]的元素和

因此，求解[left，right]范围内的值，就可以看做，s[right+1] - s[left]的值（因为s[0]=0）

```java
class NumArray {
    private final int[] s;

    public NumArray(int[] nums) {
        s = new int[nums.length+1];
        for (int i = 0 ;i<nums.length;i++){
            s[i+1] = s[i] + nums[i];
        }
    }
    
    public int sumRange(int left, int right) {
        return s[right+1] - s[left];
    }
}
```
